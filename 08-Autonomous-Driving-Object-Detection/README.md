# Autonomous Driving & Tesla Autopilot Safety Analysis

## Project Overview

This capstone project was completed as part of the Professional Certificate in Artificial Intelligence & Machine Learning from IIT Kanpur.

The project consists of two independent components:

### Part 1: Autonomous Vehicle Object Detection
Develop a deep learning-based computer vision model capable of:

- Detecting vehicles from road images
- Localizing vehicles using bounding boxes
- Evaluating object detection performance
- Running inference on unseen images

### Part 2: Tesla Autopilot Safety Analysis
Perform exploratory data analysis on Tesla accident and fatality records to understand:

- Accident trends over time
- Geographic distribution of incidents
- Driver and occupant fatalities
- Pedestrian and cyclist involvement
- Autopilot-related incidents
- Vehicle model-wise event distribution

The objective is to derive insights regarding autonomous driving technologies and road safety.

---

## Business Problem

Autonomous Vehicles (AV) and Intelligent Transport Systems (ITS) rely heavily on accurate vehicle detection and safety monitoring.

This project addresses:

1. Real-time vehicle detection using deep learning.
2. Data-driven analysis of Tesla Autopilot-related incidents.
